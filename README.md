# E-Profile Website

## Overview

This project is a personal E-Profile Website developed as part of the AFE project. The website serves as an online portfolio that presents personal, academic, and professional information in a structured and visually appealing manner.

The website is designed using HTML and CSS and is hosted using GitHub Pages, making it accessible from anywhere through a live web link.

## Features

* Professional profile section
* About Me page
* Educational background
* Technical skills showcase
* Project portfolio
* Contact information
* Responsive user interface
* Live deployment using GitHub Pages

## Technologies Used

* HTML5
* CSS3
* GitHub
* GitHub Pages

## Project Objectives

* Create a professional online presence
* Showcase skills and achievements
* Learn website development fundamentals
* Understand GitHub version control
* Deploy a live website using GitHub Pages
